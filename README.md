# Coursebox
Find your next favorite course on Coursebox

## Stack
NestJS + mySQL

## Spec
### 1. Teachers: 
**A teacher** might have **n courses**.
### 2. Courses: 
**A course** is belong to **1 teacher**, have **n chapters**, classified to **n tags**

### 3. Members:
**A member** can join **0..n courses**
